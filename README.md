# koa-mvc-starter

## Use

> git clone https://github.com/qqxs/koa-mvc-starter.git

> cd koa-mvc-starter && yarn install

> yarn run build

> yarn run server

如果开发使用

> yarn run server:dev

## views

渲染模版使用`pug`
